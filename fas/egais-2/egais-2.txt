Благодарю за тщательное и всестороннее рассмотрение моего обращения в ФАС России № 52696-ЭП/17 и сообщаю следующие факты,
неизвестные ФАС России (и мне) в момент рассмотрения обращения № 52696-ЭП/17 (далее - предыдущее обращение).

В ответе на предыдущее обращение ФАС России мне было сообщено, что работа УТМ ЕГАИС возможна на операционных системах
Microsoft Windows, Ubuntu и Debian. При более детальном изучении сайта ЕГАИС мною было выяснено следующее:

1. На сайте ЕГАИС по адресу http://egais.ru/podkljuchenie_k_egais размещён порядок подключения к ЕГАИС.
В разделе "Подключение к ЕГАИС организаций и индивидуальных предпринимателей, осуществляющих розничную продажу алкогольной продукции"
пункт 4 описывает требования для входа в личный кабинет. В частности, требуется:
а) Открыть портал http://egais.ru и пройти по ссылке «Войти в личный кабинет».
б) Нажать кнопку «Ознакомиться с условиями и проверить их выполнение».

После выполнения указанных действий сайт ЕГАИС отобразил мне страницу по адресу
https://service.egais.ru/checksystem/check
на которой были указаны следующие условия:

Операционная система - Microsoft Windows XP или выше.
Установлен и корректно работает программный компонент для работы с электронной подписью с использованием Web-браузера (Фсрар:Крипто).
Проверено наличие аппаратного ключа.
Проверка наличия электронной подписи и успешности ее использования.

Ниже размещён текст:
Операционная система - Microsoft Windows XP или выше
Браузер Internet Explorer 9.0 или выше
Установлен и корректно работает программный компонент для работы с электронной подписью с использованием Web-браузера Internet Explorer ( Фсрар-Крипто 2 ).
Убедитесь, что на вашем компьютере вставлен аппаратный ключ.

Нажатие кнопки "Начать проверку" на операционной системе Ubuntu приводит к отображению визуального символа несоответствия
требованиям в виде четырёх красных крестов (см. приложение).

Таким образом, ЕГАИС требует использования проприетарного платного ПО Microsoft Windows и Microsoft Internet Explorer
для входа в личный кабинет на сайте ЕГАИС.


2. На сайте ЕГАИС по адресу http://egais.ru/tehnicheskie_sredstva в разделе
"Рекомендуемые технических характеристики программно-аппаратных средств ЕГАИС, соответствие которым обеспечит их бесперебойное функционирование, для сельскохозяйственных товаропроизводителей и крестьянских (фермерских) хозяйств"
указаны, в частности:
а) Операционная система	от Microsoft Windows 7 Professional 64X и выше;
б) СУБД	от Microsoft SQL Server Express Edition 2008 и выше
в) Общесистемное ПО	Microsoft.NET Framework 3.0

В разделе "Рекомендуемое оборудование ЕГАИС для производителей и импортеров этилового спирта алкогольной и спиртосодержащей продукции"
указаны, в частности:

В подразделе "Сервер базы данных":
Операционная система	от Microsoft Windows Server 2003 Standard Edition R2 и выше
СУБД	от Microsoft SQL Server Standard Edition 2005 и выше
Общесистемное ПО	Microsoft.NET Framework 3.0

В подразделе "Совмещенный сервер приложений и сервер обмена данными"
Операционная система	От Microsoft Windows Server 2003 Standard Edition R2 и выше
Общесистемное ПО	Microsoft.NET Framework 3.0

В подразделе "АРМ оператора (совмещенное АРМ администратора, АРМ учета марок и оборота ЭС, АП, ССП и АРМ штрихкодирования)":
а) Операционная система	от Microsoft Windows XP Professional SP3 и выше
б) Офисное ПО	от Microsoft Office 2000 и выше


Таким образом, несмотря на то, что УТМ ЕГАИС вляется кроссплатформенным и способен к работе
на операционных системах, отличных от Microsoft Windows, ЕГАИС предположительно принуждает к приобретению
операционной системы Microsoft Windows и браузера Microsoft Internet Explorer, являющихся платным ПО,
для использования личного кабинета на сайте ЕГАИС.
